# Bolten Spray — Packaging Cosmético 2026

Plataforma de catálogo interactivo de packaging cosmético para Bolten Spray S.A.S.

## Cómo desplegar

### GitHub + Vercel (actualización automática)

1. Sube este proyecto a un repositorio en GitHub
2. Entra a [vercel.com](https://vercel.com) e importa el repositorio
3. Vercel detecta automáticamente el `vercel.json` y despliega
4. Cada vez que hagas un push a GitHub, Vercel actualiza el link automáticamente

### Estructura
```
bolten-project/
├── index.html     ← Plataforma completa (todo incluido)
├── vercel.json    ← Configuración de despliegue
└── README.md      ← Este archivo
```

## Tecnologías
- HTML5 / CSS3 / JavaScript vanilla
- Fuentes: Playfair Display + Plus Jakarta Sans (Google Fonts)
- Sin dependencias externas — todo autocontenido
